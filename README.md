### svg2font

```
  npm install svg2font --g
```

### js
```
  const svg2font = require('svg2font')
  svg2font(options)
```

### command

```
  svg2font
```

### options

|  props  | defaultValue |  description  |
| :---- | :------ | :---- |
| src | '' | svg dir |
| formats | ['svg', 'ttf', 'woff', 'woff2', 'eot'] | font type|
